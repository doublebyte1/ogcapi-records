|===
^|*Requirement {counter:req-id}* |*/req/local-resources-catalog/created*

When included in the information model of a local resources object, The value of the `created` property SHALL conform to the date-time production of https://tools.ietf.org/html/rfc3339#section-5.6[RFC 3339].
|===
